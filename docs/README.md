## 🎯 按场景查找文档

### 场景 1: 我想开发一个新的应用

1. 阅读 [5分钟快速接入指南](QUICKSTART.md)
2. 在 Casdoor 创建应用
3. 集成登录功能
4. 开始使用 API 存储数据

### 场景 2: 我需要实现用户登录

1. 阅读 [开发者接入指南 - 用户认证](DEVELOPER_GUIDE.md#一用户认证接入)
2. 在 Casdoor 创建应用
3. 集成 OAuth 登录
4. 使用 JWT Token 调用 API

### 场景 3: 我需要存储和查询数据

1. 阅读 [开发者接入指南 - 数据库操作](DEVELOPER_GUIDE.md#二数据库-crud-操作)
2. 定义 `app_identifier` 和 `collection_type`
3. 使用 API 创建、查询记录

### 场景 4: 我需要上传文件

1. 阅读 [开发者接入指南 - 文件管理](DEVELOPER_GUIDE.md#三文件管理)
2. 小文件使用直接上传
3. 大文件使用预签名上传

### 场景 5: 我需要部署到生产环境

1. 阅读 [部署文档](../DEPLOYMENT.md)
2. 修改 `.env` 配置
3. 启动服务
4. 配置监控和备份

### 场景 6: 我想查询某个 API 的用法

1. 阅读 [API 参考手册](API_REFERENCE.md)
2. 找到对应的 API 端点
3. 查看请求/响应示例

## 💡 学习路径建议

### 初级开发者

```
1. 5分钟快速接入指南 (了解基本概念)
2. API 参考手册 (熟悉 API 接口)
3. 动手实践 (开发第一个应用)
```

### 中级开发者

```
1. 开发者接入指南 (掌握完整功能)
2. API 参考手册 (深入学习 API)
3. 项目架构文档 (了解后端设计)
```

### 高级开发者

```
1. 项目架构文档 (深度理解架构)
2. 部署文档 (掌握生产部署)
3. 源代码阅读 (参与开发)
```
